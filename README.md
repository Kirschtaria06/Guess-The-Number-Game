# Number Guessing Game

A simple JavaScript-based number guessing game where players try to guess a number between 1 and 100. The game includes multiple difficulty levels, input validation, and dynamic clues for an interactive experience.

---

## Features

- Three difficulty levels: Easy, Medium, Hard  
- Input validation to prevent invalid guesses  
- Attempt tracking with different limits per difficulty  
- Clues provided in Hard mode  
- Replay functionality without refreshing the page  

---

## How to Play

1. Open the HTML file in your browser.  
2. Choose a difficulty level:  
   - Easy: 10 attempts  
   - Medium: 5 attempts  
   - Hard: 1 attempt with a clue  
3. Enter a number between 1 and 100 when prompted.  
4. The game will guide you by telling you if your guess is too high or too low.  
5. Continue guessing until you find the number or run out of attempts.  
6. After the game ends, you can choose to play again.  

Open the HTML file in your browser to start playing.
